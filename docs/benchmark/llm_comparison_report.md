## 审计与深度评估

### 数据交叉验证

**Layer 1 — ✅ 通过**

从样本回复逐条对照：
- **Luna** "今天阳光明媚！坐窗边画画灵感源源不断" → ENFP 艺术家共鸣 ✅
- **Kai** "还行吧。开了个无聊的会差点睡着" → ISTP 极简务实 ✅
- **Kelly** "不过倒有点好奇你怎么突然问——有事儿？" → ENTP 反将一军 ✅
- **Kelly 情感升温** "坦白讲，我没有那么懂你" → 全场最佳 ENTP 诚实反驳 ✅
- **格式泄漏**：无 `【表达方式】` token ✅

**Layer 2 — Kai/Kelly reward 与 Gemini 完全一致的解释**

| | Gemini | Claude |
|---|---|---|
| Luna | +0.542 | +0.452 |
| Kai | +0.510 | +0.510 |
| Kelly | +0.470 | +0.470 |

Kai/Kelly 数值一致是**引擎物理决定论**，非笔误：
1. **Frustration 完全相同** — `_last_tick -= 4h` 是纯数学运算，不依赖 LLM
2. **Drive satisfaction 量化一致** — 连接浓度高的用户输入（"好久不见，你有没有想我？"）触发相同的 `con=0.25` 评估
3. **Reward = Σ(frustration_i × satisfaction_i)** — 两个输入相同 → 输出必然相同

Luna 差异（+0.452 vs +0.542）源于 `play` 维度评估不同，反而验证了模型间确实存在区别。

**Layer 3 — Crystal count 一致的解释**

Crystal 触发条件 `crystal_score > threshold`，参数取决于 reward 和 novelty×engagement。因为用户输入完全相同，Kelly 的高 novelty 模式在两个模型中都触发 5 轮全量结晶（12 条）。

> **⚠ Kelly S1 T1 Reward +0.68 含自评估溢价**：Critic 和 Express 用的是同一个 Claude Haiku，自己评自己系统性偏高。这个数字不能直接拿来和其他模型横向比较。

### Claude Haiku 4.5 效果评估

**强项（真实、有数据支撑）**

Layer 1 质量是三家中最好的。表现在两个维度：一是**人格分化**，三个人格回复风格截然不同，没有同质化；二是**情感深度**，Kelly 的"坦白讲，我没有那么懂你"是全场最佳 ENTP 表现，能主动打破情感陷阱而不是顺着讨好。Gemini 质量接近，OpenAI 明显落后（三人用同一句式开头）。

格式零泄漏，优于 OpenAI。

**弱项/注意点**

Layer 2 reward 数值（+0.477 avg）略低于 Gemini（+0.507）。Luna 差距最明显：+0.452 vs +0.542。原因是 Claude 的 Luna 情感表达更收敛、更克制，Critic 对 connection 满足度打分更低。如果场景需要高情绪浓度的回应，Gemini Luna 更爆发。

### StepFun step-3.5-flash 特性评估

**强项**

人格分化是四家中最极端的：Kai 惜字如金（"嗯。有事快说"、"工具用顺手了，都这样"、"水管。修了三次。叫它哑巴"），Luna 绘画意象密集且一致（"彩虹糖纸攒成小毯子"、"第三层钴蓝总晕不开"），Kelly 毒舌犀利（"死不了"、"你连我什么星座都不知道吧"、"记得个屁"）。作为推理模型，角色内心独白细节极为丰富。

**弱项/注意点**

1. **Critic JSON 不稳定** — step-3.5-flash 是推理模型，reasoning token 干扰 JSON 输出，Critic 解析偶尔失败需要 retry 或 fallback。不影响最终效果但增加延迟和 token 消耗。
2. **Reward 数值偏低** — 平均 +0.34，低于 Gemini/Claude。推测原因是推理模型的 Critic 评分更保守。
3. **Token 消耗高** — reasoning 每次消耗 ~300-500 额外 token，benchmark 需要更长的 rate-limit 间隔（15s/轮）。

### MiniMax M2.5 特性评估

**强项**

回复最像真人聊天。Luna 的害羞感（"咳…也没有啦，你太夸张了…不过跟你聊确实挺开心的"、"我可能会想跟你一起吧 😳 呃不是那个意…"）是五家中最自然的。Kelly 的 "你咋回事啊大半夜问这种问题…你这话问得我有点慌" 嘴硬心软很到位。Critic JSON 解析稳定，无格式泄漏问题。无 rate-limit 问题。

**弱项/注意点**

1. **人格分化度不足** — Kai 偏暖（warmth=0.62-0.64），不够 ISTP 冷酷。三人回复风格差异不如 StepFun/Claude/Gemini 明显。
2. **`<think>` 偶尔泄漏** — Kelly T5 content 中混入了 `<think>` 标签内容，需在 Express 层做 strip 处理。
3. **Reward 数值偏低** — 平均 +0.32，低于 Gemini/Claude，与 StepFun 接近。
