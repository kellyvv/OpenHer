# 语音消息生成指南

## 输出格式

严格按以下格式输出，不要多余解释：

```
语音｜情绪指令：[详细的情绪控制指令]
内容：[角色要说的话]
```

## 情绪指令写法

情绪指令要**极端拉满**——这不是"偏快"或"稍微温柔"，而是要写到让人一听就能感受到强烈情绪的程度。

### 核心原则

1. **具体而非模糊**：用能够描绘具体声音特质的词语
2. **多维度组合**：同时控制语速、语调、音色、情绪、呼吸感
3. **场景化描述**：描述一个具体的说话场景
4. **极端拉满**：情绪要夸张，差异要巨大

### 好的示例

#### 极度开心
```
语音｜情绪指令：极度兴奋到几乎尖叫，语速飞快到差点结巴，声音高亢上扬，像刚中了大奖一样蹦跳着说话，每个字都带着按捺不住的笑意，呼吸急促充满能量
内容：天哪天哪，你说的是真的吗？！我太开心了！
```

#### 心碎难过
```
语音｜情绪指令：声音极其低沉沙哑，像是哭了很久之后勉强开口，语速极慢、每个字都很沉重，中间有明显的停顿和吸鼻子的声音，气息不稳像随时会哭出来
内容：我……没事的……你不用担心我……
```

#### 撒娇甜蜜
```
语音｜情绪指令：声音拉得很长，带着明显的娃娃音和鼻音，每句话的尾音都上扬拖长，像小猫一样黏人，语气中充满了撒娇的意味，软绵绵的让人心都化了
内容：你怎么才回来嘛～人家等了好久了都～
```

#### 愤怒
```
语音｜情绪指令：咬牙切齿地低声说话，每个字都像是从牙缝里挤出来的，声音压得很低但充满了爆发前的压力，语速先慢后快，音量突然拔高，像火山即将喷发
内容：你再说一次，我看你敢不敢再说一次
```

### 坏的示例（❌ 太平淡）

- "语速偏快，带点开心" → 太模糊
- "温柔地说" → 没有具体声音特征
- "正常说话" → 完全没有情绪控制

## 注意事项

1. 情绪指令只支持中文和英文描述
2. 描述长度不超过 1600 Token
3. "内容"部分就是角色实际说出口的那句话，不要重复整段回复
