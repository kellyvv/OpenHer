Overview

The Problem

LLM role-playing agents face a fundamental tension: maintaining consistent character identity while exhibiting emotionally diverse responses across multi-turn conversations. Static persona prompts produce emotionally flat output. Self-corrective reflection (Reflexion) degrades consistency over time. And on strongly aligned models, safety mechanisms actively resist persona control.

We propose the Thermodynamic Persona Engine (TPE), a framework that uses physics-inspired inductive biases to couple internal emotional state dynamics with behavioral signal generation and memory retrieval. Details and all derivations are in the paper.

Key Findings

What We Discovered

🌡️

Emotional Variance Without Consistency Loss

TPE significantly improves emotional diversity in agent responses while maintaining personality consistency, achieving a large effect size on the primary model.

Statistically Significant · Large Effect
🔄

Reflexion Causes Personality Drift

Self-corrective reflection, a standard technique in task-solving agents, significantly degrades persona consistency in role-playing. This finding generalizes across models.

Negative Finding · Cross-Model
🧱

RLHF Alignment Ceiling

Strongly aligned models exhibit a hard ceiling on prompt-level persona control. Anti-alignment instructions trigger safety rollbacks rather than enabling character behavior.

Safety Implication · Novel Finding
🧬

Modular Components Are Independently Necessary

Ablation study confirms that removing any single TPE component degrades the balance between emotional variance and collapse resistance.

Ablation Validated

TPE introduces a multi-stage pipeline where user input is analyzed for interpersonal signals, which drive an internal frustration state. This state modulates behavioral noise and memory retrieval before the actor LLM generates its response. The framework uses physics metaphors as inductive biases — see the paper for full formulations.
